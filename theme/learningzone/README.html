Getting Started

LearningZone is 100% responsive HTML5 and CSS3 based on bootstrap framework.

This education Moodle theme has everything you need to create an excellent education websites.

This theme comes up with numerous integrated and visual features, which make it very easy for you to establish/start your own Education Moodle Website.

This theme has powerful admin features, which allow you to customize a number of entity/item in whole site as well in single post/page and anywhere.

LearningZone Moodle theme comes with excellent support and service.

This theme is tremendously customizable in fonts, Images, banner, footer section, layout, colors, logos and many more.

LearningZone Moodle theme is compatible with Moodle version 2.7 to 3.5

This theme is 100% responsive that adapts to any screen size so it is as easy to use on mobile as it is on tablet and desktop.

Theme Installation

Installation through plugin

  • After Download folder first unzip the "learningzone-Responsive-Moodle-Theme" and you will find two zip folder one (Learningzone-for-3.1-3.2) is compatible with moodle 3.1 to 3.2 version and other(Learningzone-for-3.3) is compatible with moodle 3.3 version.
  • Login as an admin and go to Administration > Site administration > Plugin.
  • Click the link 'Install Plugin'.
  • Choose the "Plugin type" as "Theme (theme)"
  • Select any theme zip folder as per your moodle version. Upload theme zip folder or drag and drop zip folder in the "ZIP package" area.
  • Accept the statement of "Acknowledgement" and then click "Install theme from the ZIP file" button to continue.
  • Check that you obtain a 'Validation passed!' message, then click the button 'Install theme'.

Installation through FTP

  • Login into your FTP client server where your moodle is installed.
  • Now Go to Moodle "Theme" directory and place the theme folder as per your moodle version, after extracted from the Zip in it.
  • Ensure that the New Theme folder and its contents are readable by the webserver. Change the Read and Write permission of it (incorrect permission may prevent the display of newly installed theme).
  • Now login as admin to your Moodle site and go to Administration > Site administration > Notification , you will see a Notification for theme to be installed. Click "Upgrade Moodle Database now" button to continue for the installation procedure.

File Size Issue

In upload time if you get any upload size issue go to folder wamp > bin > apache > apache2.4.23 > bin > php.ini (line no 809) change upload_max_filesize = 2M to upload_max_filesize = 5M

How to setup your moodle site exactly like our demo.

If you need to setup your site exactly like our demo please follow below step.

  • After install theme go to theme settings page (Home / Site administration / Appearance / Themes / learningzone) and set everything as per your need.
  • Upload your Favicon, Logo, Change displaydashboard and other settings.
  • Upload Banner image, Internal Banner, Login background image etc... and change color option if you need.
  • In this theme have Numbers settings change background and other changes for this section if you need.
  • Change image and content for "GET THE COACHING TRAINING" section as per your need.
  • Upload Faculties images with name and descriptions.
  • Enable List of Courses, List of Categories, Announcement etc... Page path (Home / Front page settings / Edit settings)
  • You can set "footer settings" through theme settings page (Home / Site administration / Appearance / Themes / learningzone /Footer Settings)

Theme settings Options

learningzone theme is very well structure and you can customize your theme easily. No coding knowledge required.

General Settings

  • Upload your brand logo here.
  • Upload your Favicon.
  • Upload Internal Banner and login page Background image.
  • Change theme color etc...

Front Page Settings

Home / Site administration / Appearance / Themes / learningzone / Front Page Settings

  • Upload banner image.
  • Write banner heading with tagline.
  • Button text and URL
  • Upload background image for Numbers section and change icons and counts with heading
  • Change Course Heading and Course Tagline.
  • Upload Coaching background images with heading,content,button text and URL
  • Upload Faculty images with name and description.

Home / Site administration / Appearance / Themes / learningzone / Footer Settings

  • Add your social media URL.
  • Set footer content here with URL.
